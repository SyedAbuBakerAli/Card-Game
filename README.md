# Card-Game
Tech-Stack: Redis, Go, React

<p>Redis database run on docker</p>
<br>
<p>start backend server
<ul>cd backend</ul>
<ul>go run main.go</ul>
</p>
<br>
<p>start frontend server
<ul>cd frontend</ul>
<ul>npm start</ul>
</p>
